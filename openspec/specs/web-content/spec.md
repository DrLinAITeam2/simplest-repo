## Requirements
### Requirement: Static Timestamp Display
The system SHALL display a static timestamp after the "hello world" message in index.html.

#### Scenario: Timestamp added after hello world
- **WHEN** the OpenSpec proposal is implemented
- **THEN** the index.html contains "hello world" followed by a static timestamp showing the implementation time
